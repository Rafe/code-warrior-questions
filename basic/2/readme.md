#Fabonacci

Write a function to return fabonacci sequence for n

## Defination

    f(n) = f(n - 1) + f(n - 2)

when n = 0, f(n) = 0, n = 1, f(n) = 1

## Hint

Watch out when handling large number of n, how can you speed up the excution time?

## Extra

Resolve in iteration, with fewest varible.
